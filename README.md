# BoxSafe — interactive mockup

A clickable prototype of **BoxSafe**, a clinician-led medical-compliance and digital Boxer Competition Record Book platform for Australian amateur boxing. Boxing SA pilot federation.

**Live:** https://jtross00.github.io/boxsafe-mockup/

10 screens — 5 web (sign-in, governance dashboard, club admin, boxer detail/digital BCR, tournament event) and 5 mobile (boxer home, ringside doctor bout assessment, coach view, suspension lookup, audit trail). Single static `index.html`, no build, Tailwind CDN.

This is mockup only — no real data, no real backend.
